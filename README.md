# Odoo-Hackthon-2026

A static frontend prototype for an ESG-focused dashboard and collaboration portal, built with HTML, CSS, and JavaScript.

## Features

- Dashboard, Environmental, Social, Governance, Gamification, Reports, and Settings screens.
- Topbar navigation with tabbed content.
- Demo login overlay for fast access.
- Local static hosting via Python HTTP server.

## Getting Started

1. Open `Ecosphere/frontend` in your terminal.
2. Run:

```powershell
cd "c:\Odoo_hackthon\Odoo-Hackthon-2026\Ecosphere\frontend"
python -m http.server 8000
```

3. Open http://localhost:8000/index.html in your browser.

## Demo Login

Use the demo login form and click `Demo Sign-In` for instant access.
