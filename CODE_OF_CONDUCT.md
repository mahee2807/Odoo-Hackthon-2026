# Code of Conduct

## Our Pledge

In the interest of fostering an open and welcoming environment, we pledge to make participation in this project a harassment-free experience for everyone. We believe in respect, collaboration, and the free exchange of ideas.

## Expected Behavior

All contributors are expected to:

- Be respectful and considerate in all communications.
- Use welcoming and inclusive language.
- Be supportive of new contributors.
- Accept constructive feedback gracefully.
- Respect the privacy and opinions of others.

## Unacceptable Behavior

Unacceptable behavior includes, but is not limited to:

- Harassment, intimidation, or discrimination.
- Abusive or insulting comments.
- Public or private shaming.
- Posting sexually explicit or violent content.
- Personal attacks against contributors.

## Enforcement

If you experience or witness unacceptable behavior, please report it to the project maintainers by opening an issue or emailing the maintainer directly.

We will review reports promptly and take appropriate action, which may include removing comments, asking the contributor to revise behavior, or banning repeat offenders.

## Scope

This code of conduct applies to all project spaces, including GitHub issues, pull requests, commits, discussions, and any community channels associated with the project.
